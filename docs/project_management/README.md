# Project Management

Current product and contributor docs live outside this folder. Use the repo
front door and the active docs under `docs/` for the current Forge surface.

This folder preserves project-management material that should remain reachable
without competing with the canonical docs surface:

- Active branch plans, including the current bounded-work plan:
  [`plans/active/feat-bounded-work-redesign/PLAN.md`](plans/active/feat-bounded-work-redesign/PLAN.md)
- Preserved historical notes under [`history/`](history/)
- Historical implementation plans under [`plans/history/`](plans/history/)
- Architecture decision records under [`adrs/`](adrs/)
- Future backlog material under [`future/`](future/)

Directory guide:

- `adrs/`: architecture decision records
- `history/`: preserved historical docs and notes
- `plans/active/`: active branch plans once relocated from the repo root
- `plans/history/`: superseded plans kept for reference
- `future/`: backlog and future-direction material that is not part of the
  current front door
